# Problem Vault

A small knowledge base for challenging algorithmic problems.

Each problem gets its own folder. There is no fixed file layout — drop in
whatever is useful for that problem (statement, solution source, notes,
editorial, sample tests, scratch work, etc.).

## Convention

```
problem_vault/
  <problem-slug>/
    ... whatever files this problem needs ...
```

- One folder per problem, named with a short descriptive slug
  (e.g. `cf-1942-g-tree-queries` or `atcoder-abc464-d`).
- Files within a folder are freeform.
