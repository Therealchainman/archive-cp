## B. Red Pandaships

### Solution 1:  parity
```cpp
void solve() {
    int n, k;
    cin >> n >> k;
    int ans = 0;
    for (int i = 0; i < k; i++) {
        int u, v;
        cin >> u >> v;
        if (abs(u - v) % 2 == 0) ans++;
    }
    cout << ans << endl;
}

signed main() {
    int T;
    cin >> T;
    while (T--) {
        solve();
    }
    return 0;
}
```
