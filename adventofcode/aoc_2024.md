## Day 1

### Solution 1:  sorting, counter

```py
A, B = [], []
with open("big.txt", "r") as f:
    for line in f:
        left, right = map(int, line.split()) 
        A.append(left)
        B.append(right)
    A.sort()
    B.sort()
    p1 = 0
    for x, y in zip(A, B):
        p1 += abs(x - y)
    p2 = 0
    a = Counter(A)
    b = Counter(B)
    for v in A:
        p2 += v * b[v]
    print("part 1:", p1)
    print("part 2:", p2)
```
